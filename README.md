# khairi-iss-finder
 
1. Install Nodejs: https://nodejs.org/dist/v17.5.0/node-v17.5.0-x64.msi
2. Navigate to client/ folder
3. Run "npm install" 
4. Navigate to server/ folder
5. Run "npm install" then "npm run dev"
